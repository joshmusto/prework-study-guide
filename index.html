Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Basic text organization is done in HTML, including headers 1-6, paragraphs, and lists

CSS

  • A margin is the space around the outside of an element
  • A padding is the space inside an element but around the content
  • CSS can be done inline (contained within HTML) or via external stylesheet
  • Classes can be made with ".className" then defining attributes for that class within brackets

Git

  • git status: checks the current branch
  • git checkout -b branch-name: creates a new branch with "branch-name" and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "message": used after "git add -A". Associates a message with the current commit
  • git pull origin main: pulls the base branch into the current feature branch, good to use before pushing code
    • git push origin branch-name: pushes the branch "branch name"
